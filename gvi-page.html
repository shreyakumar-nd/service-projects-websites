Goodwill Notes App

This semester we collaborated with our project team to build a notes application for Goodwill. We were able to successfully connect the front-end and back-end as well as make significant progress on the front-end pages using the figma wireframes. We used Insomnia to make sure the proper data was transferred and that the connection was solid. We expanded upon the previous team's repositories by introducing several new pages using React Native.
