Add/Remove Sources

The client requested the ability to add and remove entries from the "source" database with the click of a button. We implemented buttons that trigger a popup window and corresponding database queries for the user to be able to add or remove a source from the database with ease.

Source Search Filter

The site had various source drop-downs that could be used on the site. The client wanted it to be possible to select more than one of these options at a time including the ones that could be added/removed from the ticket above

Null DOB Field

The client wanted us to fix a bug that would occur when presenting the date of birth of a user if they had yet to input a value. Before we made changes if a user did not input their date of birth, it would show up as 00-00-0000, we changed this so that it would either appear as the word Null or in other cases it appears as mm/dd/yyyy.
